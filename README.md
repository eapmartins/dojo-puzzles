DojoPuzzles
===========
Esse repositório tem o objetivo de treinamento, os casos são retirados do site DojoPuzzles.

No momento temos os seguintes scripts:
- Quebra de Linha (Java/JUnit/Maven) - http://dojopuzzles.com/problemas/exibe/quebra-de-linha/
- Ano Bissexto (Javascript/Jasmine) - http://dojopuzzles.com/problemas/exibe/ano-bissexto/
